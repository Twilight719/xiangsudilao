## 一、产品概述

**元气地牢** 是一款像素风格的 Roguelike 动作射击游戏。玩家在随机生成的地牢中探索，击杀敌人、收集武器、选择天赋，每局都是全新的体验。击败三层的 Boss 即可通关。

游戏同时支持 **桌面端（键鼠）** 和 **手机端（触控横屏）**，并通过自建账号系统实现 **云端存档同步**，玩家可跨设备续玩。

| 基本信息 | |
|---------|------|
| 产品名称 | 元气地牢（Pixel Dungeon） |
| 类型 | Roguelike / 动作射击 / 地牢探险 |
| 平台 | Web 浏览器（桌面 + 手机） |
| 技术栈 | HTML5 Canvas + Spring Boot + H2 |
| 游戏时长 | 单局约 8-15 分钟 |
| 画面风格 | 像素风（Pixel Art） |

## 三、核心特性

### 🎲 Roguelike 核心体验
- **随机地牢**：每局地牢布局、敌人分布、掉落内容完全随机
- **永久死亡**：角色死亡后本局进度清空，但宝石和升级保留
- **成长构筑**：击杀敌人获得经验升级，每次从 3 个随机天赋中选择 1 个

### ⚔️ 丰富的战斗系统
- **10 种武器** 跨越枪械、近战、法杖三大品类
- **3 级品质** 的武器 tier 系统
- **17 个天赋** 覆盖攻击、防御、移动、资源四大方向
- **8 种敌人 + Boss** 各具特色的 AI 行为
- **Boss 三阶段** 弹幕战斗

### ☁️ 账号与云同步
- 注册/登录账号系统
- 大厅存档云端同步（宝石、升级、战绩）
- 游戏中 **每 15 秒自动存档**，支持断点续玩
- 游客模式（无需注册）

### 📱 跨平台支持
- 桌面端：WASD 移动 + 鼠标瞄准射击
- 手机端：双摇杆（左移动 + 右射击）+ 功能按钮
- 自适应横屏布局，竖屏时自动提示旋转

### 🏪 局外养成
- 5 项大厅永久升级（生命、护盾、能量、冷却缩减、宝箱品质）
- 战绩统计（总场次、总击杀、最佳楼层）

## 四、游戏系统详解

### 4.1 角色与属性

玩家操控一名骑士角色，基础属性如下：

| 属性 | 基础值 | 说明 |
|------|--------|------|
| 生命值 (HP) | 6 + 大厅升级 | 归零则死亡，本局结束 |
| 护盾 | 4 + 大厅升级 | 受伤优先扣护盾，护盾自动回复 |
| 能量 | 150 + 大厅升级 × 20 | 射击消耗能量，自动回复 |
| 移动速度 | 200 px/s | 键盘/摇杆控制 8 方向移动 |
| 翻滚 | CD 2.5s | 0.35s 无敌冲刺，CD 可缩减 |
| 经验/等级 | 初始 8 exp 升级 | 每级 ×1.4，升级时选择天赋 |

### 4.2 武器系统

玩家最多携带 **4 把武器**，初始装备手枪和短刀。武器分为 3 个品类：

#### 远程枪械（消耗能量）

| 武器 | Tier | 基础伤害 | 能耗 | 射速 | 特性 |
|------|------|---------|------|------|------|
| 🔫 破旧手枪 | 1 | 3 | 1 | 0.25s | 基础均衡 |
| 🔫 突击步枪 | 1 | 4 | 2 | 0.12s | 高射速压制 |
| 🔫 霰弹枪 | 1 | 3×5 | 3 | 0.6s | 5 发扇形散射 |
| 🔫 狙击枪 | 1 | 12 | 5 | 0.8s | 单发高伤 |
| 🔫 十字弩 | 1 | 15 | 4 | 1.0s | 最高单发伤害 |

#### 近战武器（不消耗能量）

| 武器 | Tier | 基础伤害 | 攻速 | 特性 |
|------|------|---------|------|------|
| 🗡️ 短刀 | 1 | 5 | 0.35s | 快速近战 |
| 🪓 战斧 | 1 | 10 | 0.7s | 高伤慢速 |
| 🔪 匕首 | 1 | 3 | 0.15s | 极速攻击 |

#### 法杖（消耗能量，散射）

| 武器 | Tier | 基础伤害 | 能耗 | 射速 | 特性 |
|------|------|---------|------|------|------|
| 🪄 法杖 | 1 | 6×3 | 4 | 0.4s | 3 发散射 |
| 🔥 火焰法杖 | 1 | 4×5 | 5 | 0.5s | 5 发散射 |

> **武器品质**：Tier 2 伤害 ×1.6，Tier 3 伤害 ×2.3。楼层越高，高 Tier 概率越大。

| 子弹反弹 | 稀有 | 子弹碰墙反弹 1 次 |
| 火焰强化 | 稀有 | 火焰法杖弹幕 +2 |
| 淬毒之刃 | 稀有 | 近战附加 2 点中毒伤害 |

#### 🛡️ 防御天赋（3 个）

| 天赋 | 品质 | 效果 |
|------|------|------|
| 生命强化 | 普通 | +1 最大生命 |
| 护甲强化 | 普通 | +1 最大护盾，+20% 回盾速度 |

#### 💎 资源天赋（4 个）

### 4.5 地牢生成

- **网格**：5×5 网格随机游走
- **起点**：从中心 (2,2) 开始
- **房间数**：7 - 12 个房间
- **层数**：共 3 层，通过第 3 层传送门即通关
- **房间类型分配**：
  - 15% 精英房
  - 10% 宝箱房
  - 5% 商店
  - 其余为普通战斗房
  - 最远房间为 Boss 房
  - 保证至少 1 个宝箱房 + 1 个商店

### 4.6 房间与战斗机制

| 房间类型 | 尺寸 | 特点 |
|---------|------|------|
| 🏠 出生房 | 720×480 px | 低围栏装饰，无敌人 |
| ⚔️ 普通战斗房 | 720×480 px | 木桶/板条箱/椅子装饰 |
| 💀 精英房 | 720×480 px | 石柱/烛台/桌子，难度更高 |
| 👑 Boss 房 | 720×480 px | 石柱/烛台/地毯，击杀后出现传送门 |
| 📦 宝箱房 | 720×480 px | 书架/展示柜，含宝箱 |
| 🏪 商店 | 720×480 px | 长桌/椅子/烛台，含商人 NPC |

**战斗流程：**
1. 玩家首次进入房间 → 所有门关闭
2. 根据房间类型生成对应敌人
3. 击杀全部敌人 → 门打开 → 房间标记为"已清理"
4. Boss 房全部击杀 → 蓝色传送门出现（进入下一层）

### 4.7 大厅升级

使用局内获得的 **宝石** 在大厅购买永久升级：

| 升级项 | 效果 | 最高等级 | 价格递增 |
|--------|------|---------|---------|
| ❤️ 最大生命 | 每级 +1 HP | 5 级 | 递增 |
| 🛡️ 最大护盾 | 每级 +1 护盾 | 5 级 | 递增 |
| ⚡ 最大能量 | 每级 +20 能量 | 5 级 | 递增 |
| 🔄 冷却缩减 | 每级 -12% 翻滚 CD | 5 级 | 递增 |
| 📦 宝箱品质 | 提高好掉落概率 | 3 级 | 递增 |

在地牢中遇到商店房间时，按 F 键与商人交互：

| 商品 | 价格 | 效果 |
|------|------|------|
| 💚 治疗 | 30 金币 | 回复 2 HP |
| 🔵 护盾恢复 | 20 金币 | 补满护盾 |
| ⚡ 能量恢复 | 25 金币 | 补满能量 |
| 🎁 随机武器 | 50 金币 | 获得一把随机品质武器 |

### 4.9 掉落与拾取

敌人死亡掉落概率：

| 掉落物 | 概率 | 效果 |
|--------|------|------|
| 🪙 金币 | 35% | 增加金币（商店使用） |
| ⚡ 能量 | 25% | 回复能量 |
| 💚 生命 | 8% | 回复 1 HP |
| 💎 宝石 | 5% | 增加宝石（大厅升级使用） |

**武器掉落**：宝箱和敌人有概率掉落武器。拾取距离 120px，磁吸天赋可扩展至 180px。

### 4.10 存档系统

#### 大厅存档（永久）

| 存档内容 | 存储位置 |
|---------|---------|
| 宝石数量 | localStorage + 云端 |
| 5 项升级等级 | localStorage + 云端 |
| 战绩统计 | localStorage + 云端 |

#### 单局存档（中途）

| 存档内容 | 存储位置 |
|---------|---------|
| 当前楼层 | localStorage + 云端 |
| 击杀数 / 金币 / 宝石 | localStorage + 云端 |
| 玩家完整状态（JSON） | localStorage + 云端 |
| 房间布局（JSON） | localStorage + 云端 |

**存档策略：**
- 游戏中每 **15 秒** 自动保存
- 暂停退出 / 关闭页面前自动保存
- 死亡或通关后自动删除单局云端存档
- 首次加载时云端与本地取较新值合并

## 六、账号与云同步

### 账号功能

| 功能 | 说明 |
|------|------|
| 注册 | 用户名 3-32 字符，密码 4-64 字符 |
| 登录 | JWT Token 认证，有效期 24 小时 |
| 游客模式 | 无需注册，数据仅存本地 |
| 退出登录 | 清除本地 Token，数据保留 |

### 云端同步

```
本地 localStorage ←→ 云端 H2 数据库
       ↓                    ↓
  离线可玩            在线自动同步
```

- **登录后自动同步**：进入大厅时拉取云端最新数据
- **实时保存**：购买升级、宝石变动即时同步
- **跨设备续玩**：手机登录同一账号 → 恢复对局进度
- **安全机制**：密码 BCrypt 加密存储，API 带 JWT Token 认证

## 八、部署与运行

### 环境要求

| 依赖 | 版本要求 |
|------|---------|
| JDK | 17 或更高 |
| 操作系统 | Windows 10+ / macOS / Linux |
| 浏览器 | 任意现代浏览器（Chrome / Edge / Firefox / Safari） |

### 启动方式

**方式一：一键启动（推荐）**
```
双击 start-game.bat → 自动启动后端 + 打开浏览器
```

**方式二：命令行启动**
```bash
cd backend/target
java -jar dungeon-backend-1.0.0.jar
# 浏览器访问 http://localhost:8080
```

**关闭服务**
```
Ctrl + C（正常关闭，安全断开数据库）
```

### 手机访问
1. 电脑和手机连接 **同一 WiFi**
2. 启动后端后，手机浏览器访问 `http://<电脑IP>:8080`
3. 如无法访问，需在 Windows 防火墙放行 8080 端口

## 九、数据存储

## 十、游戏流程

```
开始游戏
   │
   ├─ 加载资源（精灵图、音效）
   │
   ▼
主菜单
   │
   ├─ 注册 / 登录 / 游客模式
   │
   ▼
游戏大厅
   │
   ├─ 查看宝石、战绩
   ├─ 购买永久升级（生命/护盾/能量/CD/宝箱）
   │
   ▼
进入地牢（新游戏 或 恢复存档）
   │
   ▼
┌─────────────────────────┐
│  探索房间 → 击杀敌人     │
│    ↓                    │
│  获得经验 → 升级选天赋   │
│    ↓                    │
│  拾取金币/武器/宝石      │
│    ↓                    │
│  找到传送门 → 下一层     │
│    ↓                    │
│  第 3 层 → 击杀 Boss     │
│    ↓                    │
│  🏆 通关！              │
└─────────────────────────┘
   │
   ▼
结算界面（战绩 + 宝石奖励）
   │
   ▼
返回大厅 → 购买升级 → 再次挑战
```
